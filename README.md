# InTheKitchen.com
> A sample site for learning [Asp.net core](https://learn.microsoft.com/en-us/aspnet/core/?view=aspnetcore-7.0).

## Description
Building a e-commerce site with asp.net core 7 razor pages, and javascript.

## Overview
The architectural approach for this project is based on the “Clean Architecture” design by 
[Robert C Martin (Uncle Bob)](https://blog.cleancoder.com/uncle-bob/2012/08/13/the-clean-architecture.html).
An illustration of this design paradigm is shown below:
![alt text](git-assets/clean-code-img.jpg)
